# Datasets

All datasets go in this folder (`learning/datasets/`).
Do NOT commit dataset files to git — they are in `.gitignore`.

## Primary Dataset — Stack Overflow Q&A

**Used on:** Days 3–23 (attention visualization, embeddings, RAG, coding assistant, AgentOps Platform)

**Kaggle:** [StackSample: 10% of Stack Overflow Q&A](https://www.kaggle.com/datasets/stackoverflow/stacksample)

**Files to download:**
- `Questions.csv`
- `Answers.csv`
